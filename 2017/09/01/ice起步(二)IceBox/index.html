IceBox 可以代替通常的server,方便进行很多ice服务管理，

服务器 接口方法实现接上一篇。

1，实现com.zeroc.IceBox.Service，完成方法。

2 配置文件 config.properties

可以一个配置文件里配置多个服务，如上面的最后三句。
3 启动：main class com.zeroc.IceBox.Server 添加参数：
